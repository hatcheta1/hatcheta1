## Projects

### [Recess Pieces: Can Our Behavior Predict a Recession?](https://github.com/uchicago-2026-capp30122/project-recess-pieces)

Compared "traditional" recession indicators, such as unemployment rates, to "nontraditional" indicators such as consumer sentiment and Google Trends top search queries. Then, built an interactive dashboard to visualize comparisons between traditional and nontraditional recession indicators, including a linear probability model to discover if we can predict a recession based on the distribution of categories in Google Trends top queries.

### [Queer Motherland Website](https://queermotherland.org/)

Using Ruby on Rails, I built the website for Queer Motherland, a Texas-based nonprofit organization that aims to uplift Black, queer people across the diaspora through education and community-building. This website holds information about the organization's mission and programs for Queer Motherland's 14.7k Instagram followers, and those who encounter them at events.
